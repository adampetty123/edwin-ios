# Edwin (iOS, native Swift)

Native SwiftUI rewrite of the Edwin app: unified WhatsApp + iMessage inbox with an assistant that triages, summarizes, and drafts. Human approves every send.

- SwiftUI, iOS 17+, zero external dependencies
- Auth: Supabase (GoTrue REST via URLSession), session in Keychain
- Project generated by XcodeGen (`project.yml`); CI builds on GitHub Actions macOS runners and ships straight to TestFlight

## CI secrets required
- `DIST_P12_B64`, `DIST_P12_PASSWORD` — distribution certificate
- `PROVISIONING_PROFILE_B64` — App Store profile for com.flowjam.iris
- `ASC_API_KEY_B64`, `ASC_KEY_ID`, `ASC_ISSUER_ID` — App Store Connect API key
